
# BigBrain.
A ReactJS Webapp that can identify faces in a given image and display the result.


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

ReactJS, Bootstrap, Clarifai API
